## 一. 查看函数命令

### 1.1 Mac/Linux 环境查看函数

```python
pydoc function  // 具体参考函数
```

### 1.2. Windows 环境查看函数

```python
python -m pydoc function  // 具体参考函数
```

### 1.3. 阅读源码命令

阅读命令：

```python
python3 -m pydoc -p 8081
```

自己封装了一个脚本，调用命令：

```python
source ./doc/pydoc.py
```
